# ⌨️ Vernaveg: Typing Beyond Limits

🚀 Typing Beyond Limits — Learn Faster, Type Smarter

---

## 📌 Introduction
Vernaveg is a smart desktop application developed to enhance typing speed, accuracy, and efficiency. It provides an interactive platform with structured typing practice, audio-based learning, and real-time performance tracking.

The application combines learning with engagement through practice modes, challenges, and analytics, helping users improve their typing skills effectively.

---

## 🎯 Objective
- Improve typing speed and accuracy  
- Provide engaging and structured typing practice  
- Track user performance and progress over time  
- Enable admin-based monitoring and management  
- Deliver a user-friendly and efficient typing experience  

---

## 🌐 Application Domain
Educational learning software with gamified typing practice.

---

## ✨ Features

### 👤 User Features
- ⌨️ Real-time typing practice  
- 🔊 Audio-based typing (listen and type)  
- 🎮 Practice mode and game mode  
- 📊 Speed (WPM) and accuracy tracking  
- 📈 Progress history and statistics  
- 👤 User profiles with saved preferences  

---

### 🔐 Admin Features
- Secure login with username and passkey  
- Access to all user data  
- Monitor typing performance of all users  
- View statistics, progress, and activity  
- Centralized data management system  

---

## 🛠️ Tech Stack
- **Language:** Python 3  
- **GUI:** Tkinter, CustomTkinter  
- **Audio:** Text-to-Speech (TTS)  
- **Visualization:** Matplotlib  
- **Utilities:** NumPy  
- **Database:** SQLite (sqlite3)  
- **Content Files:**  
  - `sentences.txt`  
  - `practice_words.txt`  

---

## ⚙️ How It Works
1. User logs into the system  
2. Selects typing mode (practice/test/audio/game)  
3. System provides text or audio input  
4. Tracks typing in real time  
5. Calculates:
   - Words Per Minute (WPM)  
   - Accuracy  
6. Saves user progress and performance data  
7. Admin can access and monitor all user data
